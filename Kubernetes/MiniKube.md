#### In Minikube, a Kubernetes cluster is composed of one VM. A cloud is composed of one or more VM(s)

<br/>

### <b>Start Minikube</b>

```bash
$ minikube start
```

### <b>Check Minikube status</b>

```bash
$ minikube status
```

### <b>Get pods</b>

```bash
$ kubectl get pods
```

### <b>Check nodes</b>

```bash
$ kubectl get nodes
```

### <b>K8S dashboard</b>

```
$ minikube dashboard
```
